# Gemini Context & Guardrails (Portfolio)

このリポジトリは Astro を使用した個人ポートフォリオ兼ブログです。

## 🚨 Critical Mandates

1.  **NO Tailwind CSS**: スタイリングは CSS Modules (`.module.css`) または Vanilla CSS を使用してください。
2.  **Astro Standards**: `src/pages` によるファイルシステムベースのルーティングに従い、コンポーネントは Atomic Design に基づいて `src/components` 内に配置してください。
3.  **TypeScript Strictness**: 型安全性を重視し、可能な限り明示的な型定義を行ってください。
4.  **Testing & Validation**: 重要なロジックやコンポーネントの修正後は、ビルドおよびテスト (`npm run build`, `npm run test`) を実行して整合性を確認してください。
